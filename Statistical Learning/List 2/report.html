In this task we will examine different calculations of the Prediction Error of models of various sizes.

Subtask 1: For each of the considered models

a)

Estimate ββ with the Least Squares method and calculate residual sum of squares and the true expected value of the prediction error.

  • As one can see looking at the table below, RSS is getting smaller with the model size. This is as expected because the model is able to overfit to the data.
b)

Use the residual sum of squares to estimate PE assuming that σ is known and replacing σ with its regular unbiased estimator.

  • When calculating PEPE for known σσ I used the following formula: PE=RSS+2∗σ∗pPE=RSS+2∗σ∗p
  • When calculating PEPE for unknown σσ I used the following formula: PE=RSS+2∗MSE∗pPE=RSS+2∗MSE∗p

PE for unknown σ performed much better, because it wasn’t shrinking. And we know that including more than 5 first features introduces only redundant variables to our model. Therefore we would expect the PE to stay at the same level for number of columns greater than 5.

c)

  • When calculating PE using CV I used the following formula: PE=CV=∑ni=1(Yi−ˆYi1−Mii)2

Here we can see that the errors were cumulating when the model size was growing. This method becomes pretty unstable for bigger models (> 500 columns in our case).
